Dependencies And Extras
=======================

Runtime dependencies:

- `PyYAML`

The `dev` extra installs the local contributor toolchain:

- `build`
- `mypy`
- `pre-commit`
- `pytest`
- `pytest-cov`
- `ruff`
- `sphinx`
- `sphinx-rtd-theme`
- `twine`
- `uv`

Install with:

.. code-block:: bash

   pip install -e ".[dev]"
